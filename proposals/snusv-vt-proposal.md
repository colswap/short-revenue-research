## 아이템 2: 레주메랩 (ResumeAI) — AI 이력서·포트폴리오 피드백 서비스

**주요타겟/채널**
상반기 공채·인턴 지원 중인 대학생 (취준생) / 에브리타임 취업 게시판, 취준 카카오 오픈채팅방, 블라인드

**BM(수익모델)**
건당 서비스료 — 일반(AI 분석 리포트) 3만원 / 프리미엄(AI + 선배 1:1 Zoom 30분) 8만원

**객단가**
3만원 (일반) / 8만원 (프리미엄)

**예상수익**
- 3주 목표: 일반 80건 + 프리미엄 15건
- 매출: (80 × 3만원) + (15 × 8만원) = 240만원 + 120만원 = **360만원**
- 비용: GPT-4o API 약 10만원, 서버 5만원 = 15만원
- **순이익: 345만원**
- D+3부터 첫 주문 수령 가능

**MVP 소요시간**
2일 (GPT API 연동 백엔드 1일 + 결제 링크·서비스 소개 페이지 Notion 1일)

**전략적 본질/경쟁력**
- **이식 사례**: Babson $10 Challenge (Case #47) — 인턴 지원 마감 직전 캠퍼스에서 이력서 검토 서비스로 D+1에 $1,270 달성 / Transistor.fm (Case #51) — 제품 완성 전 선판매로 14일 내 $3,750
- 3월 = 삼성·LG·SK 등 대기업 상반기 서류 접수 시즌 + 스타트업 인턴 공고 시즌
- "AI 초안 분석 + 서울대 선배 검수"라는 이중 검증 포지셔닝 → 타 AI 서비스 대비 신뢰도 우위
- 에브리타임은 이미 모여있는 취준생 포획 시장 (마케팅 비용 0원)
- 왜 서울대 팀이 해야 하는가: "서울대 선배 피드백"이라는 브랜드가 가격 프리미엄의 핵심 정당화 요인

**실행 로드맵**
| 일자 | 핵심 액션 |
|------|---------|
| D+1 | GPT-4o 이력서 분석 프롬프트 완성 (직무별 5개 템플릿), 카카오페이/토스 결제 링크 생성 |
| D+2 | Notion 서비스 소개 페이지 완성, 에브리타임 서울대·연세대·고려대·KAIST 취업 게시판 동시 포스팅 |
| D+3 | 첫 주문 수령, 2~4시간 내 회신 보장 → 후기 스크린샷 확보 |
| D+4~7 | 후기 2~3개 확보 후 재홍보, 취준 카카오 오픈채팅방 20곳에 후기 기반 홍보 |
| D+8 | 프리미엄 패키지(Zoom 1:1) 추가 런칭, 단가 상승 |
| D+9~14 | 선착순 5명 20% 할인 이벤트로 2차 수요 자극 |
| D+15~21 | 서류 마감 막바지 홍보 집중 ("오늘 마감인데 이력서 망했다면") — 긴급성 트리거 |

**비고**
- 홍보 채널: 에브리타임 5개 대학, 블라인드 취준 게시판, 취준 카카오 오픈채팅방(상반기 공채 스터디 검색)
- 롱테일 키워드: "서울대 선배 이력서 피드백", "AI 이력서 검토", "공채 자소서 첨삭"

**객단가**
프로젝트당 150~300만원 (1~2주 작업 기준)

**예상수익**
- 3주 목표: 2건 계약
- 매출: 평균 200만원 × 2건 = **400만원**
- 비용: 서버(AWS 프리티어/Google Colab 무료) 0원, 홍보 0원
- **순이익: 400만원**

1일 (포트폴리오 PDF 2장 + LinkedIn 프로필 정비 + 제안서 템플릿)

**비고**

## 아이템 5: 신학기 AI 강의 가이드 — "서울대생이 직접 뽑은 강의 TOP 20" 유료 정보 리포트

**주요타겟/채널**
복수전공·부전공 고려 중인 서울대 재학생, 신입생 / 에브리타임 서울대 게시판, 학과 단톡방, 인스타그램 서울대 계정

**BM(수익모델)**
리포트 단건 구매 (5,000원) + 교육 스타트업 스폰서십 (30~50만원/건)

**객단가**
5,000원/건 (리포트) + 스폰서십 40만원/건

**예상수익**
- 리포트: 300건 × 5,000원 = 150만원
- 스폰서십: 1~2건 × 40만원 = 40~80만원
- 비용: GPT API 2만원, Canva Pro 0원(무료 티어) = 2만원
- **순이익: 188~228만원**
- D+3부터 첫 매출 발생 가능

**MVP 소요시간**
2일 (설문·데이터 수집 1일 + 리포트 디자인 제작 1일)

**전략적 본질/경쟁력**
- **이식 사례**: NomadList (Case #45) — "구글 스프레드시트 → 유료 멤버십 전환", D+3에 $566 달성 / Stanford $5 레스토랑 예약팀 (Case #11) — 정보 비대칭을 이용한 즉각 현금화
- 3월 초 수강변경 기간(통상 개강 후 1~2주)이 포획 시장: 강의 정보에 목마른 학생들이 특정 시기에 집중
- "에브리타임 강의평 읽기 귀찮다" → 큐레이션·정리만으로도 지불 의사 발생
- 왜 서울대 팀이 해야 하는가: 서울대 내부인만 접근 가능한 강의 데이터 + 서울대생 신뢰 네트워크가 진입장벽

**실행 로드맵**
| 일자 | 핵심 액션 |
|------|---------|
| D+1 | 구글 폼 설문 배포 ("서울대 명강의 추천", 재학생 100명 목표), 에브리타임 수작업 데이터 수집 병행 |
| D+2 | ChatGPT로 강의 카테고리별 분석 초안, Canva로 10페이지 PDF 리포트 디자인 |
| D+3 | 에브리타임 서울대 자유 게시판에 "무료 샘플 1페이지" 선공개 + 전체 리포트 카카오페이 유료 판매 시작 |
| D+4~7 | 학과 단톡방에 리포트 홍보, 수강변경 기간 내 집중 판매 |
| D+8 | 에듀테크 스타트업 5곳에 스폰서십 제안 Cold Email 발송 |
| D+9~14 | 수강변경 기간 마지막 날 "오늘까지 할인" 긴급성 마케팅 |
| D+15~21 | 스폰서십 확정, 다음 학기·여름학기 시즌 예약 판매 선착순 모집 |

**비고**
- 주의: 에브리타임 자동 크롤링은 ToS 위반 가능 → 수작업 수집 또는 직접 설문으로 대체 필수
- 홍보 채널: 에브리타임 서울대, 학과 카카오 단톡방, 서울대 인스타그램 계정 협업 요청
- 롱테일 키워드: "서울대 명강의 추천", "수강변경 강의 정보", "서울대 A+ 강의 목록"
- 리스크: 수강변경 기간 이후 수요 급감 → 타이밍이 핵심. D+1부터 즉시 실행하지 않으면 기회 소멸
- 시즌 한계 보완: 주기적 업데이트 구조 (매 학기 신규 리포트)로 반복 수익 가능

## 아이템 6: 1주 MVP 팩토리 — 비개발 창업팀 대상 MVP 개발 패키지 (고단가 지위 게임 핵심)

**주요타겟/채널**

200~500만원/팀 (패키지 규모에 따라: 랜딩페이지 80만원 / 웹앱MVP 250만원 / 풀스택MVP 500만원)

**예상수익**

**MVP 소요시간**

**전략적 본질/경쟁력**
